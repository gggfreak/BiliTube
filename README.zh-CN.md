# BiliTube

一个致力于**提升 B站 (Bilibili) 播放体验**并提供**硬字幕遮挡**功能的浏览器插件。

BiliTube 包含两大核心功能：

1. **B站播放增强**：为你带来类似 YouTube 的丝滑操作体验（例如长按鼠标或空格键实现 2倍速 播放）。
2. **听力专注（字幕遮罩）**：当视频自带硬字幕时，练听力很容易不知不觉变成“读字幕”。BiliTube 可以在字幕上方加一个可自由拖拽、缩放的遮罩层，强迫你专注依赖声音和画面去理解。

## 这是什么

BiliTube 是一个轻量级浏览器插件，旨在提供更纯粹的视频观看和外语学习体验。

它主要做两件事：

- **优化控制**：在 Bilibili 等网站注入更好用的播放快捷操作（如长按倍速）。
- **视线遮挡**：在页面上添加一个高度可定制的本地遮罩层，用于遮挡不需要的画面元素（主要是硬字幕）。遮罩层可自由拖拽、缩放、调整透明度，并能按站点记忆位置。

---

## 适合什么场景

- 想要在 B站 (Bilibili) 获得类似 YouTube 的长按倍速沉浸体验
- 英语听力练习
- 不想总是下意识看字幕
- 想保留画面，只遮住字幕区域
- 某些网站字幕直接烧在画面里，无法关闭

---

## 主要功能

- 可拖拽字幕遮罩
- 可从四角调整大小
- 可调透明度
- 按网站保存位置和尺寸
- **Bilibili 专属增强**：
    - **长按加速播放**：在 Bilibili 视频页面，长按鼠标左键或空格键可实现 2 倍速播放，松开恢复。
    - **智能区分短按**：短按鼠标左键或空格键仍为 Bilibili 原生播放/暂停功能。
    - **加速视觉提示**：加速时，视频顶部中央会显示“2x ⏩”的胶囊型提示。
    - **YouTube 风格快捷键**：
        - `k`：播放/暂停
        - `j` / `l`：快退 / 快进 10 秒
        - `0`~`9`：快速跳转到视频 0% ~ 90% 的进度
        - `Shift` + `p` / `n`：播放上一个 / 下一个视频
        - `f` / `t` / `i`：切换 全屏 / 网页全屏 / 画中画模式
        - `m`：切换静音

## 使用方法

1. 打开一个视频页面
2. 点击插件图标，开启遮罩
3. 将遮罩拖到字幕区域
4. 从四角拖动以调整大小
5. 在弹窗中调整透明度
6. 双击遮罩可恢复默认位置

### 绑定快捷键

如果你希望使用快捷键快速开关遮罩，可以在浏览器中手动绑定：
1. 在浏览器地址栏输入 `chrome://extensions/shortcuts` 并回车。
2. 找到 `BiliTube` 扩展。
3. 点击快捷键输入框，在键盘上按下你想要的快捷键组合（例如 Mac 下推荐使用 `Command + Shift + M` 或 `Option + X`）。

## 全屏说明

不同网站的视频播放器实现不同，因此全屏行为不完全一致。

常见情况：

- 有些网站的原生全屏会让普通网页遮罩层无法显示
- 某些网站需要使用特定的容器全屏方案，才能让遮罩保持可见
- 因此在不同站点上，表现可能略有差异

这属于浏览器和网站播放器结构带来的正常限制。

## 权限说明

为了最大程度地保护用户隐私和安全，BiliTube 使用了 `activeTab` 权限模型。这意味着在您通过点击扩展图标或使用快捷键明确激活它之前，本插件无权访问任何网站。

当您激活它时，它会获得对**当前标签页**的**临时权限**，从而能够注入字幕遮罩。这种模式既保证了插件可以在任何视频网站上灵活使用，又比请求“所有网站”的永久访问权限安全得多。

对于 Bilibili 网站，插件请求了特定的主机权限，以便播放增强功能可以自动加载，提供无缝的用户体验。

主要权限说明：
- `storage`: 用于在本地保存遮罩的透明度、位置等设置。
- `activeTab`: 响应您的点击操作，临时获取当前页面的权限以注入遮罩。
- `scripting`: 执行脚本以注入遮罩 UI 和播放增强功能。
- `bilibili.com` 的主机权限: 允许 B站 专属增强功能自动加载。

## 隐私与安全边界

BiliTube 的定位是一个**本地、单一用途工具**。

它不会做这些事：

- 不登录账号
- 不做统计追踪
- 不做云同步
- 不执行远程代码或远程规则
- 不上传页面内容、字幕内容、视频内容或浏览历史

它仅在浏览器本地保存少量配置，例如：

- 遮罩位置
- 遮罩尺寸
- 透明度
