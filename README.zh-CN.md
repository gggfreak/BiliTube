# Subtitle Masker

一个用于**遮挡视频硬字幕**的浏览器插件。

适合这样的场景：当字幕直接显示在画面里时，练听力很容易不知不觉变成“读字幕”。Subtitle Masker 会在字幕区域上方加一个可拖拽、可缩放的遮罩层，帮助你更专注地依赖声音、语境和画面去理解内容。

> English README: [README.md](./README.md)

## 极简安装说明

1. 下载或克隆本仓库
2. 打开 Chromium 系浏览器扩展管理页
3. 打开 **开发者模式**，点击 **加载已解压的扩展程序**
4. 选择本项目文件夹
5. 打开视频页面，点击插件并启用遮罩

正常限制：
- 某些网站在全屏下表现会不一样
- 某些播放器需要你手动重新拖一下遮罩位置
- 目前这是通过 GitHub 分发、以开发者模式加载的扩展，不是 Chrome 商店版本

---

## 这是什么

Subtitle Masker 是一个轻量级浏览器插件，用来遮挡视频里的硬字幕区域。

它不是视频处理工具，也不会修改视频文件本身。它做的事情很简单：

- 在页面上添加一个本地遮罩层
- 可以拖动到字幕位置
- 可以调整大小和透明度
- 按站点记住位置和尺寸

---

## 适合什么场景

- 英语听力练习
- 不想总是下意识看字幕
- 想保留画面，只遮住字幕区域
- 某些网站字幕直接烧在画面里，无法关闭

---

## 主要功能

- 可拖拽字幕遮罩
- 可从四角调整大小
- 可调透明度
- 按网站保存位置和尺寸
- 快捷键开关：`Alt+S`

## 使用方法

1. 打开一个视频页面
2. 点击插件图标，开启遮罩
3. 将遮罩拖到字幕区域
4. 从四角拖动以调整大小
5. 在弹窗中调整透明度
6. 双击遮罩可恢复默认位置

---

## 全屏说明

不同网站的视频播放器实现不同，因此全屏行为不完全一致。

常见情况：

- 有些网站的原生全屏会让普通网页遮罩层无法显示
- 某些网站需要使用特定的容器全屏方案，才能让遮罩保持可见
- 因此在不同站点上，表现可能略有差异

这属于浏览器和网站播放器结构带来的正常限制。

---

## 权限说明

这个插件需要较宽的网站访问权限，原因是它必须直接运行在视频页面上，才能把遮罩放到字幕区域。

不同网站的播放器结构差异很大：

- 有些是普通 video 元素
- 有些使用自定义 fullscreen 容器
- 有些视频放在 frame 里
- 有些页面会动态切换播放器结构

因此插件需要在不同视频页面中注入本地遮罩 UI。

---

## 隐私与安全边界

Subtitle Masker 的定位是一个**本地、单一用途工具**。

它不会做这些事：

- 不登录账号
- 不做统计追踪
- 不做云同步
- 不执行远程代码或远程规则
- 不上传页面内容、字幕内容、视频内容或浏览历史

它仅在浏览器本地保存少量配置，例如：

- 遮罩位置
- 遮罩尺寸
- 透明度

---

## 已知限制

- 不同视频站的兼容性表现不完全一致
- 某些特殊播放器可能需要额外适配
